# 콘솔 환경에서 구현한 스네이크 게임

## 주요 기능
- 콘솔 기반 스네이크 게임 구현
- 방향키를 이용한 뱀 이동
- 먹이 생성 및 섭취 처리
- 먹이 섭취 시 뱀 길이 증가
- 벽 충돌 및 몸통 충돌 처리
- 모든 빈 공간을 채운 경우 승리 처리

## 조작 방법
↑ ↓ ← → : 뱀 이동 방향 변경

## 실행 환경
- C#
- Windows Console
- Visual Studio 2022

## 실행 방법
- Visual Studio 2022에서 프로젝트를 열고 빌드 후 실행합니다.

## 개발 목적
- C# 기본 문법, 클래스 분리, 연결 리스트를 활용한 뱀 몸체 관리와 게임 루프 내 상태 변화 처리를 연습하기 위해 제작했습니다.
